What is DOM?

The Document Object Model (DOM) is a programming interface for web documents.
It represents the structure of a document as a tree of objects, where each node is an object representing a part of the document

How do you select an element from the dom

1. getElementById: Selects an element by its
const element = document.getElementById('myElementId');
2. getElementsByClassName: Selects all elements with a specific class name.
const elements = document.getElementsByClassName('myClassName');
3.getElementsByTagName: Selects all elements with a specific tag name.
const elements = document.getElementsByTagName('div');

What is event delegation in the context of the Dom, why it is useful?

Event delegation is a technique in JavaScript that leverages event bubbling to handle events at a higher level in the DOM tree than the element where the event originated.
Instead of adding event listeners to multiple child elements, you add a single event listener to a parent element. This parent element can then manage all events that bubble up from its child elements.

How do you manipulate an element's attributes and style using DOM?

Manipulating Attributes 1.Get an Attribute:
let element = document.getElementById('myElement'); let attributeValue = element.getAttribute('attributeName');
2.Set an Attribute
element.setAttribute('attributeName', 'newValue');
3.Remove an Attribute
element.removeAttribute('attributeName');
Manipulating Styles

1.Change Inline Styles:
element.style.color = 'red';
element.style.backgroundColor = 'blue';
2.Add or Remove CSS Classes

// Add a class
element.classList.add('newClass');
// Remove a class
element.classList.remove('oldClass');
// Toggle a class
element.classList.toggle('activeClass');
